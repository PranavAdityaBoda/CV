# CV API Reference
## Modules
### `realtime_humancount.py`
This module provides functionality for real-time human counting using computer vision. Key functions/classes:
* `count_frame` — Detects humans in a frame using **HOG (Histogram of Oriented Gradients)**.
* `main` — Runs the real-time human counting application.
